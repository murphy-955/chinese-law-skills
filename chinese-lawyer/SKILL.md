# Chinese Lawyer

为中国律师从业者提供可复用的法律实务工作流与参考资源。

## 核心能力

- 合同审查、起草与谈判要点
- 法律文书（起诉状、答辩状、律师函、法律意见书等）撰写
- 法规、司法解释与案例检索策略
- 诉讼/仲裁庭审准备与证据组织
- 常年法律顾问服务与客户沟通
- 律师执业风险与利益冲突提示

## 工作流

### 1. 合同审查

1. 明确合同类型与交易背景
2. 识别核心商业条款与风险点
3. 检查主体资格、履行能力、违约责任、争议解决条款
4. 出具审查意见（修订版 + 批注/清单）

### 2. 法律文书撰写

1. 确认文书类型与适用场景
2. 梳理事实、诉讼请求/法律意见、依据
3. 引用现行有效的法律法规与司法解释
4. 校对格式、称谓、编号、签章位置

### 3. 法规与案例检索

1. 提取关键词与法律关系
2. 检索现行有效的法律法规、司法解释、地方性规定
3. 补充检索指导案例、公报案例、典型案例
4. 整理检索结论并标注效力层级
