# 法律文书模板参考

## 民事起诉状

```
原告：姓名/名称、住所地、联系方式
被告：姓名/名称、住所地、联系方式

诉讼请求：
1. ...
2. ...

事实与理由：
...

此致
XXX 人民法院

具状人（签字/盖章）：
日期：
```

## 答辩状

```
答辩人：...
被答辩人：...

针对原告提起的 ... 纠纷一案，答辩如下：
一、...
二、...

此致
XXX 人民法院

答辩人：
日期：
```

## 律师函

```
致：XXX

XX 律师事务所依法接受 ... 委托，指派本律师就 ... 事宜致函如下：

一、事实依据
二、法律分析
三、律师意见与要求
四、期限与后果

特此函告。

XX 律师事务所
律师（签章）：
日期：
```

## 法律意见书

- 首部：委托方、事项、依据
- 正文：事实概述、法律分析、结论意见
- 尾部：律师签章、日期
