# 合同实务参考

## 审查清单

- [ ] 合同主体资格与授权
- [ ] 标的、数量、质量、价款
- [ ] 履行期限、地点、方式
- [ ] 违约责任与赔偿范围
- [ ] 不可抗力与情势变更
- [ ] 争议解决（诉讼/仲裁、管辖）
- [ ] 保密与知识产权
- [ ] 合同生效、变更、解除、终止
- [ ] 签章与份数

## 常见合同类型要点

### 买卖合同

- 交付标准、验收程序
- 所有权保留、风险转移
- 质量保证期与退换货

### 借款合同

- 利率上限（LPR 4 倍）
- 还款方式与提前到期
- 担保安排

### 劳动合同

- 必备条款（合同期限、工作内容、劳动报酬等）
- 试用期、服务期、竞业限制
- 解除与终止的法定情形

### 股权转让合同

- 标的股权与对价
- 陈述与保证
- 过渡期安排
- 工商变更与税费承担
